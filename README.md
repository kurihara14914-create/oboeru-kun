# 覚えるくん

大学受験向けの暗記トレーニング用Webアプリです。スマホでの利用を前提に、単語カード作成、文字入力式の特訓、弱点復習、コンボ・スコア記録を端末内に保存します。

## スマホでの開き方

スマホで使いやすくするには、GitHub Pages、Netlify、Vercelなどに置いてURL化してください。公開URLにすると、ホーム画面に追加してアプリのように起動できます。リポジトリ名やURL用の英字名は `oboeru-kun` を使います。

### iPhone / iPad

1. Safariで公開URLを開く
2. 共有ボタンを押す
3. 「ホーム画面に追加」を選ぶ
4. 次回からホーム画面の `覚えるくん` アイコンで開く

### Android

1. Chromeで公開URLを開く
2. メニューから「ホーム画面に追加」または「アプリをインストール」を選ぶ
3. 次回からホーム画面の `覚えるくん` アイコンで開く

`file:///.../index.html` のようなローカルファイルとして開く場合、ホーム画面追加やオフラインキャッシュは安定しません。スマホ運用ではURL化する方が分かりやすいです。

## 使い方

1. 公開URL、またはPCでは `index.html` をブラウザで開く
2. 「作成/管理」で科目を選び、穴埋め単語カードを手入力またはMarkdownでまとめて追加する
3. 「今日」から復習、弱点、全単語カードのいずれかを開始する
4. 誤って追加した単語カードは「作成/管理」の単語カード管理から削除する
5. 答えを文字入力して判定する

## AIアプリとの連携

「作成/管理」画面の「AIプロンプト」ボタンで、教科書やノート写真からMarkdown単語カードを作るためのプロンプトをコピーできます。ChatGPTなどのAIアプリに写真と一緒に渡し、生成されたMarkdownをこのアプリへ貼り付けてください。

単語カード形式はすべて「表: 穴埋め文」「裏: その答え」に統一します。ユーザーは「？？？」に入る答えだけをタイピングします。
別訳や同義語がある場合は、裏側を `捨てる / 放棄する` のように区切ります。どれか1つを入力できれば正解になります。

## Markdown形式

```md
# 英単語
- abandon = ？？？: 捨てる / 放棄する
- estimate = ？？？: 見積もる / 推定する

# 地理
- 季節によって向きが変わる風を？？？という: 季節風

# 化学
- 窒素と水素からアンモニアを合成する方法を？？？という: ハーバー・ボッシュ法
```

## 保存

単語カードと記録はブラウザの `localStorage` に保存されます。サーバーやログインは使っていないため、別端末への同期はされません。
